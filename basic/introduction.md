# jsは、もともとNetscape Navigatorというブラウザーのために開発された言語

大部分がオブジェクト、つまり値や処理をまとめたもので構成されている。

大文字と小文字を区別する。例えば `name != NAME`。

## jsではstrict modeでは、`eval`や`with`といったレガシーな機能や構文を禁止している

### 例えば

#### `eval`の禁止

```javascript
eval("var x = 10; console.log(x);") // 10
```

上記だと外部からの入力をそのまま`eval`で実行すると、任意のコードが実行される可能性がある。

#### `with`の禁止

```javascript
var obj = {a: 1, b: 2};

with (obj) {
  console.log(a); // 1
  console.log(b); // 2
}
```

`with`を使うと変数のスコープが不明確になり、バグが発生しやすくなる。どのスコープの変数が参照されているのかが分かりにくくなるため、可読性やデバッグに影響を与える。

最適化の困難: `with`により、JavaScriptエンジンがスコープを特定しにくくなり、コードの最適化が妨げられる。
